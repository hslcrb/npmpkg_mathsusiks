# Contributing to MathSusiks / MathSusiks 기여 가이드

Thank you for your interest in contributing to MathSusiks!
MathSusiks에 기여해 주셔서 감사합니다!

## How to Contribute / 기여 방법

1. **Report Bugs** / **버그 보고**
   - If you find a bug, please open an issue.
   - 버그를 발견하면 이슈를 생성해 주세요.

2. **Suggest Features** / **기능 제안**
   - We welcome new ideas for mathematical functions.
   - 새로운 수학 함수에 대한 아이디어를 환영합니다.

3. **Submit Pull Requests** / **Pull Request 제출**
   - Fork the repository.
   - Create a new branch.
   - Make your changes with bilingual comments (English first, Korean second).
   - Submit a pull request.
   - 저장소를 포크하고 새 브랜치를 만드세요. 변경 사항을 추가할 때는 영어와 한국어로 주석을 작성해 주세요. 그 후 Pull Request를 제출하세요.

## Coding Standards / 코딩 표준

- Use ES Modules. / ES 모듈을 사용하세요.
- Follow the bilingual comment style. / 한영 병기 주석 스타일을 따르세요.
- Ensure all tests pass. / 모든 테스트가 통과하는지 확인하세요.
